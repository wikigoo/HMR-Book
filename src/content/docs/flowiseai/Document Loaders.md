Document loaders allow you to load documents from different sources like PDF, TXT, CSV, Notion, Confluence etc. They are often used together with [Vector Stores](https://docs.flowiseai.com/integrations/langchain/vector-stores) to be upserted as embeddings, which can then retrieved upon query.
