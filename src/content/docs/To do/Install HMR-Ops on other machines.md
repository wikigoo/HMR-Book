سیستم مدیر پروژه (مخزن خصوصی [`wikigoo/HMR-Ops`](https://github.com/wikigoo/HMR-Ops)) فعلاً فقط روی کامپیوتر محل کار نصب شده. روی **لپ‌تاپ شخصی** و **کامپیوتر آفیس** هنوز نصب نشده است (وضعیت: باز — ۲۰۲۶-۰۷-۱۱).

## مراحل نصب روی هر ماشین

```powershell
gh auth status          # اگر خطا داد: gh auth login
gh repo clone wikigoo/HMR-Ops
cd HMR-Ops
pwsh ./setup.ps1
```

اگر `setup.ps1` پیام‌های `[ok]` چاپ کرد، نصب موفق بوده.

## به‌روزرسانی‌های بعدی

نیازی به نصب مجدد نیست — چون skill همه‌چیز را زنده از GitHub می‌خواند، تغییرات سیاست‌ها (مثل ماتریس اختیارات) خودکار اعمال می‌شوند. فقط اگر خودِ فایل‌های skill یا agentها در HMR-Ops عوض شدند:

```powershell
git pull; pwsh ./setup.ps1
```

> نکته: چون HMR-Ops خصوصی است، `gh` باید روی هر ماشین لاگین باشد وگرنه clone با خطای 404 شکست می‌خورد.
